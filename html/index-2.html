Portfolio



Dans le cadre de ma formation à l'INSA de Toulouse, j'ai réalisé un portfolio. L'objectif est d'avoir une synthèse de toutes les compétences que j'ai acquises à la fin de ma formation. Ce portfolio est organisé par thème afin d'en faciliter la lecture.

Voici la maquette 2021-2022 de la formation 5-ISS ci-dessous avec les différentes unités de formation suivies dans chaque module

Dans la section ci-dessous, vous trouverez un lien vers une page expliquant ce que j'ai fait pour chaque projet/leçon. Chaque section contient un résumé qui résume le travail accompli ainsi que les livrables associés et une seconde section contenant la partie technique. Vous pourrez accéder au différents travaux sur chaque module en cliquant sur le lien "Go".

Module Smart Device

Les cours sur les dispositifs intelligents sont des cours très liés. L'objectif de ces cours est de suivre l'ensemble du processus de création d'un capteur et son déploiement.

Module Communication

L'objectif de ces cours est de comprendre tous les concepts liés à la communication dans les réseaux de capteurs : protocoles de communication pour les objets connectés, la sécurisation de ces réseaux, les réseaux émergents ainsi qu'un cours sur l'énergie pour les objets connectés.

Module Middleware and Service

L'objectif de ces cours est de se familiariser avec tous les concepts liés à l'IoT: mise en place d'une architecture pour les objets connectés, virtualisation de réseaux, organisation de projet ainsi qu'une initiation de DevOps (Jenkins, API rest, Springboot).

Module Data and Application

L'objectif de ces cours est d'acquérir des connaissances sur la manière dont nous analysons les données. Ces cours ont montré l'importance du traitement et de la représentation des données.

Module Innovative Project

L'un des projets les plus importants et les plus ambitieux de cette dernière année Innovative Smart Systems est ce projet de localisation et de communication. L'objectif de cette partie est donc de suivre pas à pas ce projet jusqu'à sa finalité.

Module Innovation and Humanity

L'objectif de ces cours est d'acquérir des connaissances sur de nombreuses théories concernant l'organisation d'un projet, la gestion des conflits et la gestion d'équipe.
